# Havok SDK – Reference Code, Public Release (2007)
Havok SDK - PUBLIC RELEASE, BUILD(#20070919)

This repository contains the **reference source code of the Havok SDK** public release owned by the SIGMA Technology Group. Havok is a physics engine widely used in games and real-time applications for **rigid-body dynamics, collision detection, and physics simulation**.  

> This repository contains **reference source code only**. No precompiled libraries, tools, or installers are included.

## Purpose of this Repository

This repo is a reference code for software preservationists. This is published here to provide public access to ones wanting to dig into Havok-powered games and applications of early 2000s. It peserves the **source code of the Havok SDK Public Release (Build #20070919)** for:

- Game preservation and research  
- Legacy project maintenance  
- Learning and experimentation with early 2000s physics engine code


## What's Included

- **Source Code**  
  Full SDK source tree for the 2007 public release.

- **Header Files**  
  All necessary headers for building the SDK from source.

- **Documentation**  
  Original API reference and technical notes included with the source code.


## Folder Structure

/src
Core Havok SDK source code
/include
SDK header files
/docs
Original API documentation


## Requirements

- Compatible **C/C++ compiler** (MSVC 2005–2008 recommended)  
- Windows XP/2003 or compatible legacy environment for building  
- Build system compatible with legacy Visual Studio project files (if included)


## Getting Started

1. Open the provided Visual Studio solution/project files (if available).  
2. Include the `include/` directory in your project header paths.  
3. Compile the source code to produce libraries for your project.  
4. Refer to `docs/` for API details and usage guidelines.
